RaiBlocks Payment/Checkout Demo

Welcome to a demo of a RaiBlocks based payment checkout system. The current demo is functional, it works using a backend of Python3/Flask with POST requests and a websocket setup to give real time requests. Currently it is set to use 'xrb' which is the smallest usable denomination of RaiBlocks

What it does: The user completes the order form, adds their details, the server combines the total amount to be paid with a transaction id into a unique identifier (using hashids). They are then directed to another dynamically generated page where they are presented with a new xrb address to which they send the payment. This page dynamically updates using websockets and when the payment is received will show this. The transaction is then closed and the details are emailed to the merchant.

What it doesn't do: This is only a simple proof of concept demo, there is zero security right now and it doesn't keep track particularly well of orders. It also doesn't yet email the merchant.

What next: I'm hoping that the amazing RaiBlocks community will take this and make it into a functioning service for merchants.
